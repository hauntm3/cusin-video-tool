# CUSINI ROYAL VIDEO TOOL

Python-утилита для работы с видео через `ffmpeg`.

Что умеет:

- Сжимать видео
- Обрезать видео по секундам
- Работать через графический интерфейс с кнопками и картинкой в шапке
- Показывать только готовые пресеты сжатия с кратким описанием по ожидаемому уменьшению размера
- Оставлять старое интерактивное `cmd`-меню как запасной режим

## Установка

```powershell
python -m pip install -r requirements.txt
```

## Быстрый старт

Запуск графического интерфейса:

```powershell
python .\compress_video.py
```

Или двойным кликом через `.bat`:

```powershell
.\run_video_tool.bat
```

В окне можно:

1. нажать `Выбрать файл...` и открыть стандартный проводник Windows
2. выбрать режим `Сжатие` или `Обрезка`
3. в режиме сжатия выбрать один из готовых пресетов
4. указать путь сохранения результата или оставить автогенерацию имени
5. нажать кнопку запуска операции

## Команды без меню

Сжатие:

```powershell
python .\compress_video.py compress ".\video.mp4"
python .\compress_video.py compress ".\video.mp4" --crf 30
python .\compress_video.py compress ".\video.mp4" --width 1280 --preset slow
```

Обрезка:

```powershell
python .\compress_video.py trim ".\video.mp4" 5 12
python .\compress_video.py trim ".\video.mp4" 10.5 18.2 -o ".\clip.mp4"
```

## Старое консольное меню

Если нужен старый сценарий с цифрами в `cmd`, он сохранен:

```powershell
python .\compress_video.py menu
```

## Примечания

- Если запускать `python .\compress_video.py ".\video.mp4"`, это по-прежнему сработает как команда сжатия.
- Для обрезки используется точное перекодирование, чтобы начало и конец клипа совпадали с заданными секундами.
- Если `ffmpeg` не установлен в системе, скрипт попробует взять его из `imageio-ffmpeg`.

## Сборка EXE

Собрать `.exe` можно так:

```powershell
.\build_exe.bat
```

После сборки готовый файл появится в папке `dist\VideoTool.exe`.
Собранный `.exe` запускает оконное приложение без консоли.
