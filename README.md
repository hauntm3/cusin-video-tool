# CUSINI ROYAL VIDEO TOOL

Простой запуск:

## 1. Установить зависимости

```powershell
python -m pip install -r requirements.txt
```

## 2. Запустить приложение

Из исходников:

```powershell
python .\compress_video.py
```

Или двойным кликом:

```powershell
.\run_video_tool.bat
```

## 3. Как пользоваться

1. Нажмите `Выбрать файл...`
2. Выберите видео в стандартном проводнике Windows
3. Выберите `Сжатие` или `Обрезка`
4. При сжатии выберите готовый пресет
5. Нажмите `Сжать видео` или `Обрезать видео`

Результат по умолчанию сохраняется рядом с исходным файлом.

## Сборка EXE

```powershell
.\build_exe.bat
```

Готовый файл появится тут:

```text
dist\VideoTool.exe
```

## Если не запускается

- Проверьте, что установлен Python
- Проверьте, что выполнилась команда `pip install -r requirements.txt`
- Если `ffmpeg` не установлен отдельно, приложение возьмет его из `imageio-ffmpeg`
